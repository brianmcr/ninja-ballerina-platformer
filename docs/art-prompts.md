# Ninja Ballerina: The Stolen Show — Art Generation Prompts

## Art Direction Summary

- **Style:** Full illustration/cartoon — vibrant, sparkly, mobile-game quality (NOT pixel art)
- **Reference:** Concept art image shows pink tutu + black ninja sleeves/corset, ribbon whip, warm sparkly pink/purple/gold palette, magical girl energy
- **Character:** Caucasian girl with brown pigtails, pink tutu, black ninja arm sleeves and lace-up corset
- **Palette:** Hot pinks, magentas, purples, golds, sparkle effects, warm tones
- **Audience:** Ages 8-12, girl-empowerment energy, magical girl meets martial arts
- **Subtitle:** "The Stolen Show" (golden ballet slippers stolen by Soggy Waffle)

## Generation Strategy

### Recommended approach:
1. **Start with the main character** — generate the idle pose reference first, get it right, then use as reference for all other poses
2. **Use the concept art image as style reference** for every generation — upload it alongside the prompt
3. **Generate poses individually**, not as a full sprite sheet (AI is better at single images)
4. **Assemble sprite sheets manually** in Photoshop/Figma/Aseprite after generating individual frames
5. **For animation, generate key poses** (contact, passing, up, down for run cycle) then tween or fill in-betweens

### Tools:
- **DALL-E 3** or **Midjourney v6** for initial illustration generation
- **Remove.bg** for background removal
- **Aseprite** or **Piskel** for sprite sheet assembly and animation timing
- **TexturePacker** for packing sprite sheets for Kaplay

### Integration with Kaplay:
Once sprite sheets are ready, replace the `rect()` calls in player.ts with:
```typescript
loadSprite("ballerina", "sprites/ballerina.png", {
  sliceX: 6, sliceY: 7,
  anims: {
    idle: { from: 0, to: 3, speed: 4, loop: true },
    run: { from: 6, to: 9, speed: 8, loop: true },
    jump: { from: 12, to: 14, speed: 6 },
    // etc.
  }
})
```
